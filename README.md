# za-fe-cli

> 支持拉取及生成种子项目，开发，与输出生产环境的代码

### 安装
```
npm i za-fe-cli -g

```
> window平台请使用管理员权限安装，mac平台请在命令前面加上sudo

也可作为项目试玩直接拉取
```
git clone git@github.com:MapleShaw/za-fe-cli.git
cd za-fe-cli
npm i
npm link
```
### 使用
```
za add|a 添加模版
za list|l 查看模版列表
za delete|d 删除模版
za init|i 初始化项目

```
